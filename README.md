 # AgriRent

AgriRent is a full-stack web application for renting agricultural equipment and machinery.

## Features

* Equipment rental system
* User authentication
* Farmer and owner dashboard
* Booking management
* Secure payments

## Tech Stack

* Frontend: React.js
* Backend: Node.js + Express.js
* Database: MongoDB

## Project Structure

* client/ → Frontend
* server/ → Backend

## Installation

```bash
git clone <repository-url>
cd agrirent
```
